**Prerequisites:** 

1. Clone the repository using the project's Git URL 
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Create an `.env.local` file and set the right environment variables

```
VITE_BASE44_APP_ID=your_app_id
VITE_BASE44_APP_BASE_URL=your_backend_url

e.g.
VITE_BASE44_APP_ID=cbef744a8545c389ef439ea6
VITE_BASE44_APP_BASE_URL=https://my-to-do-list-81bfaad7.base44.app
```

Run the app: `npm run dev`

Open [Base44.com](http://Base44.com) and click on Publish.

# Standalone mode (NO Base44)

This export was converted to run completely outside Base44.

## Quick start (one command dev)
1) Install everything:
   npm run install:all

2) Run both server + client:
   npm run dev

- Client: http://localhost:5173
- API:    http://localhost:5050

The client calls /api/* and Vite proxies to the API.

## Data storage
The backend stores data in: server/data/db.json
(You can back it up / move it like a normal file.)
